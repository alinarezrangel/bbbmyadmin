Servidor Sesiones Base de datos Big Blue Button Usuarios


Es el puerto donde bbbmyadmin va a iniciarse. De manera predeterminada es 4567. Nota: no recomendamos utilizar puertos ya utilizados por otros programas como el puerto 80 dado que puede crear un bloqueo. De cambiar el puerto, no se veran cambios hasta el reinicio de bbbmyadmin.



El secreto de la sesión es una secuencia de caracteres alfanuméricos que se utilizarán como llave para generar las ID de sesión. Recomendamos altamente que no comparta este código y que sea lo más aleatorio posible. De manera predeterminada es 454RR5E45R345FDES.



El nombre de la base de datos de bbbmyadmin. De manera predeterminada es bbbmyadmin.



El nombre del usuario de la base de datos que utilizara bbbmyadmin. Debe poder seleccionar, insertar y actualizar las tablas. De manera predeterminada es bbbmyadmin.



Contraseña del usuario de la base de datos. De manera predeterminada, es a123b456.



Host donde se ubica la base de datos. De manera predeterminada es http://localhost/.



Secreto de Big Blue Button, puedes obtenerlo con el comando bbb-conf --secret. Es realmente importante que nunca compartas esta clave, de lo contrario podrian acceder totalmente al sistema. No posee valor predeterminado, debe fijarse antes de utilizar el sistema.



Host donde esta Big Blue Button, no solo especifique el host, sino también la ruta a la API: su tu Big Blue Button esta en http://www.example.com/bigbluebutton/, entonces este campo debe tener el valor http://www.example.com/bigbluebutton/api/. No olvide agregar una barra al final. Su valor predeterminado es http://localhost/bigbluebutton/api/.
